# Blinkit Sales Analytics Dashboard

An interactive Excel dashboard analyzing sales performance, outlet metrics, and item distribution for Blinkit - India's Last Minute App.

##  Dashboard Overview

This dashboard provides comprehensive insights into:
- **Total Sales**: $1.20M across all outlets
- **Average Sales**: $141 per transaction
- **Total Items**: 8,523 items analyzed
- **Average Rating**: 4.0 customer satisfaction score

##  Key Features

### Sales Analysis
- Year-over-year sales trends (2011-2022)
- Peak sales identification at $204.5k in 2018
- Revenue distribution across different outlet types

### Outlet Performance
- Sales breakdown by outlet size (High, Medium, Small)
- Location-based analysis (Tier 1, Tier 2, Tier 3 cities)
- Outlet type comparison (Supermarket Type1/2/3, Grocery Store)

### Product Insights
- Item type revenue analysis across 15+ categories
- Fat content distribution (Low Fat vs Regular)
- Top-performing categories: Fruits & Vegetables ($178.1k), Snack Foods ($175.4k)

### Interactive Filtering
- Filter by Outlet Size
- Filter by Outlet Location (Tier 1/2/3)
- Filter by Item Type

##  Visualizations

- **Line Chart**: Outlet establishment trends over time
- **Donut Charts**: Fat content distribution by outlet, Outlet size distribution
- **Bar Charts**: Item type revenue, Outlet location performance, Outlet type metrics
- **KPI Cards**: Key metrics at a glance

##  Technical Details

- **Tool**: Microsoft Excel
- **File Format**: .xlsx
- **Version**: Compatible with Excel 2016 and later
- **No Installation Required**: Just download and open in Excel

##  How to Use

1. Download the Excel file
2. Open in Microsoft Excel (2016 or later)
3. Use the filter panel on the left to interact with data
4. All charts and metrics update automatically based on selected filters
